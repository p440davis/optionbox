This is an optionbox

You could think of an optionbox as a hybrid of a dropdown and a radio group, whithout the downsides.

  • Unlike a dropdowns, options are initially visible, which reduces click friction and cognitive load of making selection.
  • Unlike radio groups, optionboxes close up neatly after selection, making it easy to see which option has been selected.
  • Unlike <select> dropdowns, option styling is device and browser independent. Your CSS is in control.
