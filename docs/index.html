Optionbox.js transforms your <select> drowdowns into intuitive, accessible optionboxes that obey your styles

This is an optionbox

You could think of an optionbox as a hybrid of a dropdown and a radio group, whithout the downsides.

  • Optionboxes can initially be open, enabling quick selection.

  • Optionboxes close up neatly when selected.

  • The styling of options is independent of device and browser. Your CSS is in control.

Optionbox.js is...

  • tiny (less than 3KB)
  • compatable with all major browsers (even IE!!!)
  • just javascript (no dependencies, no conflicts)
  • customisable with your own CSS
  • free and open source

Getting started

Pop this at the bottom of your HTML, before the closing <body> tag and all your <select> dropdowns will be magically transformed.

<script src="optionbox.0.1.0.min.js"></script>

<script>
  optionbox();
</script>

Settings

select:

By default, all select elements on the page will be transformed into optionboxes.

If you only want to transform some of your dropdowns, like
 <select class="my-class"> , or  <select id="my-id">  you can pass in a css-style selector, like so:

optionbox({
  select: ".my-class, #my-id"
});

persist:

By default, options selected by the user are persisted during the browser session so that they can go back, forward and refresh the page without losing their selection.

If you do not want values to be stored in the user's browser sessionStorage you can disable this feature like so:

optionbox({
  persist: false
});
